= minitest-libnotify

Test notifier for minitest via libnotify.

Source[http://github.com/splattael/minitest-libnotify] |

== Usage

  require 'minitest/autorun'
  require 'minitest/libnotify'

== Installation

  gem install minitest-libnotify
